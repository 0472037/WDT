Web Development Technology

      本课程将从实践的角度介绍现代 Web 开发技术，主要包括基本开发工具的使用、Web 前端开发技术（HTML & CSS & Javascript）、后端开发技术（Ruby 实现）。如果时间允许可能会加入部分高级话题，如网络的基本概念、HTTP 协议简介、网络爬虫简介。建议使用 Mac / Linux 系统的学生选修。
